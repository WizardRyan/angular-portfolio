Calc Guru

  • https://calc.guru

Angular Recipe App

  • https://recipe-app-angular.firebaseapp.com
  • A website where users can sign up and post recipes for everyone to see
  • Built using Angular + Angular Material and Firebase for the database
  • Lets users sign up with their own credentials, or by using Google Oauth
  • Signed in users can comment on recipes, owners of the posted recipe can delete comments
  • Try it yourself! Click the "?" on the sign in page if you'd like more help

Eye Tracking Desktop Application

  • A desktop application that connected to an eye tracker, displayed text for a user to read, tracked the user's eye movements while they read the text, and then compiled an interactive graph of the data collected
  • I wrote this for SimSaw using Windows Forms in C#, this one was a solo project
